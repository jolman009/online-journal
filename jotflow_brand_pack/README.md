# JotFlow — Mini Brand Pack

## Color Palette (extracted from the logo)
- Ink Navy — **#191E31** (primary text, dark UI, dark icon background)
- Deep Ink — **#25283C** (alternate dark, borders)
- Jot Teal — **#26BAB2** (accent, checks, CTAs)
- Ice Teal — **#88C5C6** (hover/secondary accent)
- Mist — **#D5E7E8** (subtle backgrounds, dividers)
- White — **#FFFFFF** (light UI base)

### Suggested pairs
- Ink Navy text on White background
- White text on Ink Navy background
- Jot Teal as accent on either White or Ink Navy

## Files
- `logo_lockup_light.png` — transparent background
- `logo_lockup_mono_black.png`, `logo_lockup_mono_white.png`
- `icon_mark.png` — transparent background
- `app_icon_light_*.png`, `app_icon_dark_*.png` — multiple sizes

## Basic usage rules
- Keep clear space around the mark (≈ the height of the check).
- Avoid shadows/gradients; the mark is meant to be crisp and flat.
